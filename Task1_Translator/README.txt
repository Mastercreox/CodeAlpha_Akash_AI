Language Translation Tool (Python)

Description:
This project is a simple language translation tool that converts text from any language into Hindi using the deep-translator library.

Steps to Run:
1. Install dependency:
   pip install deep-translator

2. Run the program:
   python translator.py

Usage:
- Enter any text in any language.
- The program will automatically detect the source language.
- The translated output will be displayed in Hindi.

Example:
Input: Hello, how are you?
Output: नमस्ते, आप कैसे हैं?

Conclusion:
This project demonstrates how language translation can be implemented using Python and APIs.
